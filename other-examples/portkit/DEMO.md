# PortKit On-Stage Demo Script (3 min)

> Run this against `pnpm build && pnpm start` (NOT `pnpm dev`).
> Manual smoke checklist: §12.3 of the Notion plan.

## Setup
- Browser at http://localhost:3000
- DevTools closed (open if Q&A needs network/console)
- Chat input focused, history cleared
- Speakerphone ready for jokes

## Turn 1 — 0:00
- Type: **"What's going on this week?"**
- Expect: project-dashboard surface (4 KPI cards, sprint progress bar, 3 ProjectCards: Atlas / Orion / Lyra) within ~6s.
- Talk to: "PortKit is showing me everything live — KPIs and per-project status, all generated by the agent on demand. None of this UI was pre-built; it came down the wire as an A2UI envelope."

## Turn 2 — ~0:30
- Type: **"Drill into Orion."**
- Expect: project-detail surface (status hero, milestones, kanban scoped to Orion's sprint-22 tasks, open risks).
- Talk to: "Same agent, different surface. The kanban template-binds to a filtered task list — that's A2UI v0.9's `componentId + path` pattern."

## Turn 3 — ~1:00
- Type: **"Who's overloaded?"**
- Expect: team-load surface (BarChart of points per person + DataTable with Amir + Chen flagged).
- Talk to: "The agent computed load server-side, sent a tiny envelope, the chart rendered from React."

## Turn 4 — ~1:30
- Type: **"Draft a status update for Orion."**
- Expect: status-report-draft surface (TL;DR card + Progress + Risks + Asks, two buttons at the bottom).
- Talk to: "Editable draft. The Send button has an `action.event` that bubbles back to the agent."

## Turn 5 — ~2:15
- Click the **Send to #orion-pm** button.
- Expect: button flips to "Sent" state; agent acknowledges in chat within ~4s.
- Talk to: "And there we go — the action event went back to the agent, it acknowledged, and we shipped a fake update to a fake channel."

## Fallback Turn 6 (if Q&A time)
- Type: **"Show me recent updates."**
- Expect: update-feed surface with 3 items.

## Recovery
- If Gemini stalls > 5s on any turn: "The great thing about A2UI is the streaming surfaces — even slow turns paint progressively." Then retry.
- If a surface renders blank: open DevTools → check network panel for `/api/copilotkit/*` POST response shape.
- If Send button doesn't bubble the event: check the manual checklist in §12.3 turn 5 of the Notion plan.
